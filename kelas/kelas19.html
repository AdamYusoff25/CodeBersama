Akses Array Multi-Dimensi dengan Indeks

Satu cara untuk memahami Array Multi-Dimensi ialah dengan menaggapnya sebagai array di dalam array. Semasa anda mengunakan [] untuk mencipta array anda, anda boleh menggunakan lagi satu [] untuk memasukkan array didalam array tersebut
Contoh: 

                        const Nombor = [[100, 101, 102], 200, 300];

lihat array Nombor mempunyai array [100, 101, 102] didalam arraynya.
Dan jika anda mahu mengambil sesuatu nilai dari array yang berada didalam array anda perlu mengunakan dua []
Contoh jika anda maku memangil nilai 101 dari array Nombor: 

                        Nombor[0][1]

[0] digunakan untuk mengambil nilai pertama didalam array Nombor iaitu [100, 101, 102] dan [1] digunakan untuk mengambil nilai kedua didalam array tersebut.
