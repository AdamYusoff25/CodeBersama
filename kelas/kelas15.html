Gunakan Notasi Kurungan untuk Mencari huruf dalam Ayat

Notasi kurungan adalah cara untuk mendapatkan huruf pada indeks tertentu dalam rentetan.
Kebanyakan bahasa pengaturcaraan moden, seperti JavaScript, tidak bermula mengira pada 1 seperti manusia. Mereka bermula pada 0.
Contohnya, huruf pada indeks [0] dalam perkataan Berbasikal adalah B. Jadi, jika const Hobi = "Berbasikal", anda boleh mendapatkan nilai huruf pertama ayat String dengan menggunakan Hobi[0].
Contoh: 

                        const Hobi = "Berbasikal";
const HurufPertama = Hobi[0];

Untuk mencari huruf terakhir ayat String, guna [Pembolehubah.length - 1]
Contoh

                    const Hobi = "Berbasikal";
const HurufTerakhir = Hobi[Hobi.lenght - 1];
