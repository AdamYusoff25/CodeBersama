Simpan lebih dari satu nilai dalam Satu Pembolehubah Menggunakan Array JavaScript

Dengan pembolehubah array JavaScript, kita boleh menyimpan beberapa data dalam satu pembolehubah.
Anda memulakan deklarasi array dengan tanda kurung siku buka, mengakhiri dengan tanda kurung siku tutup, dan meletakkan koma di antara setiap entri, seperti ini: 

                        const Burger = ["sayur", "daging ramly", "roti"];
Nota: Array boleh diletakkan di dalam array lain.
