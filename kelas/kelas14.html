Menyusun Ayat String dengan Pembolehubah

Membina Rentetan dengan Pembolehubah Kadang-kadang anda perlu membina ayat dengan menggunakan operator tambah +, anda boleh memasukkan satu atau lebih pembolehubah ke dalam rentetan yang anda bina. Contoh: 

                        const ContohPembolehubah = "Adam";
const contohString = "Hi, nama saya" + namaSaya + ", selamat berkenalan";

contohString sekarang mempunyai nilai

                    Hi, nama saya Adam, selamat berkenalan

Nota: Berhati-hati dengan ruang kosong. Penggabungan tidak menambahkan ruang di antara rentetan yang dirangkakan, jadi anda perlu menambahkannya sendiri.
