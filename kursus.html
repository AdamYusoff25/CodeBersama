Kursus Asas JavaScript

JavaScript ialah bahasa skrip yang anda boleh gunakan untuk menjadikan halaman web interaktif. Ia adalah salah satu teknologi teras web, bersama-sama dengan HTML dan CSS, dan disokong oleh semua pelayar moden.

Dalam kursus ini, anda akan mempelajari konsep pengaturcaraan asas dalam JavaScript. Anda akan bermula dengan struktur data asas seperti nombor dan rentetan. Kemudian anda akan belajar untuk bekerja dengan tatasusunan, objek, fungsi, gelung, pernyataan if/else dan banyak lagi.
